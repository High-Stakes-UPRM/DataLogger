# High Stakes DataLogger

Pin Set-Up:

### Temperature/Humidity Sensor   
+ PLUS  ->  5V
+ out   ->  2 (DHTPIN)
+ MINUS ->  GRD      

##### Important Sensor Information
+ Humidity Range: 20-90% RH
+ Humidity Accuracy: ±5% RH
+ Temperature Range: 0-50 °C
+ Temperature Accuracy: ±2% °C

### Micro SD Module              
+ CS    ->   10
+ SCK   ->   13
+ MOSI  ->   11
+ MISO  ->   12
+ VCC   ->   5V
+ GRD   ->   GRD
